йЎөйқўзҪ®жҚўз®—жі•жҰӮеҝө

  • еҠҹиғҪ
    • еҪ“еҮәзҺ°зјәйЎөејӮеёёж—¶ йңҖи°ғе…Ҙж–°йЎөйқўдё”еҶ…еӯҳе·Іж»Ў зҪ®жҚўз®—жі•йҖүжӢ©иў«зҪ®жҚўзҡ„зү©зҗҶйЎөйқў
  • и®ҫи®Ўзӣ®ж Ү
    • е°ҪеҸҜиғҪең°еҮҸе°‘йЎөйқўи°ғе…Ҙи°ғеҮәж¬Ўж•°
    • жҠҠжңӘжқҘдёҚеҶҚи®ҝй—® жҲ– зҹӯжңҹеҶ…дёҚи®ҝй—®зҡ„йЎөйқўи°ғеҮә
  • йЎөйқўй”Ғе®ҡ (Frame locking)
    • жҸҸиҝ°еҝ…йЎ»еёёй©»еҶ…еӯҳзҡ„йҖ»иҫ‘йЎөйқў
    • ж“ҚдҪңзі»з»ҹзҡ„е…ій”®д»Јз Ғ
    • иҰҒжұӮе“Қеә”йҖҹеәҰзҡ„д»Јз Ғе’Ңж•°жҚ®
    • йЎөиЎЁдёӯзҡ„й”Ғе®ҡж Үеҝ—дҪҚ (Lock bit)

еҰӮдҪ•и§ЈеҶіеҶ…еӯҳз©әй—ҙдёҚеӨҹз”Ё?

  • иҰҶзӣ–(Overlay)

еә”з”ЁзЁӢеәҸ жүӢеҠЁ жҠҠйңҖиҰҒзҡ„жҢҮд»Өе’Ңж•°жҚ®дҝқеӯҳеңЁеҶ…еӯҳдёӯ (е°Ҷд»Јз ҒеҲҶжҲҗиӢҘе№ІжЁЎеқ— еҪјжӯӨж— и°ғз”Ёе…ізі»зҡ„ е°Ҷе…¶йҷ„еәёеҲ°дёҖеқ—еҢәеҹҹ йңҖиҰҒз”ЁжҲ·иҮӘе·ұжқҘзЎ®е®ҡдҪ•ж—¶е°Ҷе…¶жЁЎеқ—еҠ е…ҘзЁӢеәҸ ејҖеҸ‘йҡҫеәҰй«ҳ)

  • дәӨжҚў(еҜ№жҚў Swapping)

ж“ҚдҪңзі»з»ҹ иҮӘеҠЁ жҠҠжҡӮж—¶дёҚиғҪжү§иЎҢзҡ„зЁӢеәҸдҝқеӯҳеҲ°еӨ–еӯҳдёӯ (еӣ дёәеҜ№жҚўзҡ„жҳҜдёҖдёӘиҝӣзЁӢзҡ„ж•ҙдёӘең°еқҖз©әй—ҙ жүҖд»ҘејҖй”ҖдјҡжҜ”иҫғеӨ§)

  • иҷҡжӢҹеӯҳеӮЁ

еңЁжңүйҷҗе®№йҮҸзҡ„еҶ…еӯҳдёӯ д»ҘйЎөдёәеҚ•дҪҚиҮӘеҠЁиЈ…е…ҘжӣҙеӨҡжӣҙеӨ§зҡ„зЁӢеәҸ

ж‘ҮдёҖж‘ҮзӯҫеҲ°жҳҜеҹәдәҺ iBeacon(дҪҺеҠҹиҖ—и“қзүҷйҖҡдҝЎ)

зі»з»ҹеҗҜеҠЁ

CPUеҠ з”өд№ӢеҗҺ еҲқе§ӢеҢ–еҜ„еӯҳеҷЁ д»Һ CS:IP = 0xF000:FFF0 жү§иЎҢз¬¬дёҖжқЎжҢҮд»Ө зі»з»ҹеӨ„дәҺе®һжЁЎејҸ ж•… PC = 16 * CS + IP жӯӨж—¶зү©зҗҶең°еқҖдёә 0xFFFF0 20дҪҚең°еқҖжҖ»зәҝ еҸҜз”ЁеҶ…еӯҳдёә 1MB жӯӨж—¶и·қзҰ» 1MB еҸӘеү©дёӢ 16дёӘеӯ—иҠӮ

16дёӘеӯ—иҠӮеӨҹз”Ёеҗ—?

0xffff0 зү©зҗҶең°еқҖеӨ„дёәи·іиҪ¬жҢҮд»Ө дјҡй•ҝи·іиҪ¬еҲ° BIOS д»Јз ҒзңҹжӯЈејҖе§Ӣзҡ„ең°ж–№

BIOS еҲқе§ӢеҢ–

  • еҹәжң¬иҫ“е…Ҙиҫ“еҮә
  • зі»з»ҹи®ҫзҪ®дҝЎжҒҜ
  • ејҖжңәеҗҺиҮӘжЈҖ
    1. зЎ¬д»¶иҮӘжЈҖPOST

      • жҹҘжүҫе№¶жү§иЎҢжҳҫеҚЎзӯүжҺҘеҸЈзҡ„еҲқе§ӢеҢ–зЁӢеәҸ
