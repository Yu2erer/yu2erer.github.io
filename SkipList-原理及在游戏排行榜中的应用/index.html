еҸ‘иЎЁдәҺ 2020-09-12 | жӣҙж–°дәҺ 2020-10-31 | еҲҶзұ»дәҺ жёёжҲҸи®ҫи®Ў | иҜ„и®әж•°пјҡ | йҳ…иҜ»ж¬Ўж•°пјҡ

иҝ‘жңҹе·ҘдҪңзҡ„еҶ…е®№дё»иҰҒжҳҜи®ҫи®ЎдёҖдёӘйҖҡз”Ёзҡ„жёёжҲҸжҺ’иЎҢжҰңжңҚеҠЎеҷЁ, йҖҡз”Ёзҡ„еҗ«д№үжҢҮзҡ„жҳҜеҗҢж—¶дёәеӨҡдёӘйЎ№зӣ®з»„жҸҗдҫӣдёҖеҘ—йҖҡз”Ёзҡ„и§ЈеҶіж–№жЎҲ, з»ҸиҝҮдёҚж–ӯзҡ„и°ғз ”, жңҖз»ҲеҶҷдёӢжӯӨж–Ү.

жң¬ж–Үе°Ҷдјҡи®Іиҝ° жҺ’иЎҢжҰңзҡ„еӨҡз§Қе®һзҺ°ж–№ејҸ, дёҚиҝҮжңҖз»ҲжҲ‘йҖүжӢ©дәҶ SkipList дҪңдёәеә•еұӮзҡ„ж•°жҚ®з»“жһ„, еӣ жӯӨдјҡзқҖйҮҚи®ІSkipList.

е…ҲжқҘи®ІдёҖдёӢжҺ’иЎҢжҰңзҡ„дё»иҰҒеҠҹиғҪ.

жҺ’иЎҢжҰңдё»иҰҒеҠҹиғҪ

  • жӣҙж–°/еҲ йҷӨ/иҺ·еҸ– дёҖдёӘ Key еңЁжҹҗдёӘ жҺ’иЎҢжҰң дёҠзҡ„ жҺ’еҗҚ, еҸҠеұ•зӨәдҝЎжҒҜ
  • иҺ·еҸ– дёҖдёӘжҺ’иЎҢжҰң д»»ж„ҸеҢәй—ҙ(дҫӢеҰӮжҺ’еҗҚеҢәй—ҙ) зҡ„еұ•зӨәдҝЎжҒҜ

жҺ’иЎҢжҰңжҺ’еәҸж–№жЎҲ

жёёжҲҸдёӯзҡ„жҺ’иЎҢжҰңзҡ„жҺ’еәҸж–№жЎҲж— йқһе°ұдёӨз§Қ

  • е®һж—¶жҺ’еәҸ, еҚізҺ©е®¶ж•°еҖјеҸҳеҢ–зҡ„зһ¬й—ҙ, е°ұе®ҢжҲҗдәҶеҜ№е®ғзҡ„жҺ’еәҸ
  • йқһе®һж—¶жҺ’еәҸ, дәҰеҸҜз§°д№Ӣдёәе®ҡж—¶жҺ’еәҸ, зҺ©е®¶ж•°жҚ®еҸҳеҠЁдёҚз«ӢеҚіжҺ’еәҸ, иҖҢжҳҜзӣҙеҲ°зӯ–еҲ’жүҖй…Қзҡ„ж—¶й—ҙзӮ№еҲ°зҡ„ж—¶еҖҷжүҚиҝӣиЎҢжҺ’еәҸ

иҝҷдёӨз§ҚжҺ’еәҸж–№жЎҲжІЎжңүеҘҪеқҸд№ӢеҲҶ, дё»иҰҒжҳҜзңӢзӯ–еҲ’д»–жғіиҰҒд»Җд№Ҳ? (жҲ–и®ёд»–иҮӘе·ұд№ҹдёҚзҹҘйҒ“жғіиҰҒд»Җд№Ҳ, д»–йғҪжғіиҜ•иҜ•зңӢ, д»–еҸӘйЎҫзқҖд»–иҮӘе·ұ)

ж•°жҚ®з»“жһ„зҡ„жҠүжӢ©

йҰ–е…Ҳ, еӣ дёәжҲ‘д»¬иҰҒжҸҗдҫӣз»ҷеӨҡдёӘйЎ№зӣ®з»„дёҖеҘ—и§ЈеҶіж–№жЎҲ, жҜҸдёӘйЎ№зӣ®з»„йңҖиҰҒзҡ„жҺ’еәҸж–№жЎҲеӨ§жҰӮзҺҮжҳҜдёҚеҗҢзҡ„, еӣ жӯӨ жҺ’йҷӨдәҶе®ҡж—¶жҺ’еәҸ, еҰӮжһңдёҖејҖе§Ӣе°ұеҫҖе®ҡж—¶жҺ’еәҸзҡ„жҖқи·ҜдёҠжғі, еҲ°ж—¶еҖҷе“ӘдёӘйЎ№зӣ®з»„зӘҒз„¶иҜҙиҰҒе®һж—¶жҺ’еәҸ, ж”№еҠЁиө·жқҘдјҡйқһеёёз—ӣиӢҰ, иҖҢдё”е®ҡж—¶жҺ’еәҸе®һзҺ°иө·жқҘжҜ”иҫғз®ҖеҚ•, зӣҙжҺҘеҲ°зӮ№еҝ«жҺ’е°ұе®ҢдәӢдәҶ.

йӮЈд№Ҳ жҺ’йҷӨдәҶ е®ҡж—¶жҺ’еәҸ, е°ұеҸӘиғҪжңқзқҖе®һж—¶жҺ’еәҸзҡ„жҖқи·ҜдёҠиө°.

дёҡз•ҢдёҖиҲ¬йҮҮз”ЁдёӨз§Қж•°жҚ®з»“жһ„иҝӣиЎҢе®һзҺ°

  • RB Tree зәўй»‘ж ‘
    • дјҳеҠҝ:
      • ж—¶й—ҙеӨҚжқӮеәҰзҡ„еёёж•°зӣёеҜ№ SkipList дҪҺ
      • еҶ…еӯҳеҚ з”Ёд№ҹзӣёеҜ№иҫғдҪҺ
    • дёҚи¶і:
      • е®һзҺ°еӨҚжқӮ
      • дёҚиғҪж–№дҫҝзҡ„иҺ·еҸ–еҲ°жҹҗдёӘеҢәй—ҙзҡ„дҝЎжҒҜ
  • SkipList и·іиЎЁ
    • е’Ң зәўй»‘ж ‘ зӣёеҸҚ

жҲ–и®ё е Ҷд№ҹеҸҜд»ҘеҠ иҝӣеҺ», еҸ–topK, дҪҶжҳҜи¶…еҮәдәҶkйЎ№, е°ұдёҚзҹҘйҒ“жҺ’еҗҚ.

SkipList еңЁ redis зҡ„жңүеәҸйӣҶеҗҲдёӯд№ҹжңүз”ЁеҲ°, иҝҷж—¶еҖҷеҸҜиғҪдјҡжңүдәәй—® дёәд»Җд№ҲдҪ дёҚиҖғиҷ‘зӣҙжҺҘз”Ё redis е‘ў?

жҲ‘иҖғиҷ‘зҡ„зӮ№дё»иҰҒжңүд»ҘдёӢеҮ зӮ№

  1. еј•е…Ҙ redis дјҡеўһеҠ йғЁзҪІйҡҫеәҰ
  2. еј•е…Ҙ redis д»…д»…еҸӘз”ЁжқҘеҒҡжҺ’еәҸ, жҳҜдёҚжҳҜеӨӘз¬ЁйҮҚдәҶ?
  3. redis зҡ„й»ҳи®Ө SkipList е®һзҺ°жҳҜд»Һе°ҸеҲ°еӨ§, иҖҢжҲ‘д»¬зҹҘйҒ“дёҖиҲ¬жҳҜеҲҶеҖји¶Ҡйқ еүҚзҡ„и¶Ҡз»Ҹеёёи®ҝй—®, еҰӮжһңд»Һе°ҸеҲ°еӨ§ жҹҘжүҫйқ еүҚзҡ„ж•ҲзҺҮеҸҜиғҪдёҚеҰӮд»ҺеӨ§еҲ°е°Ҹ, дҪҶжҳҜ redis еҶҷжӯ»жҺ’еәҸйЎәеәҸдәҶ, жҲ‘дёҚејҖеҝғ
  4. зҺ°еңЁжёёжҲҸжңҚеҠЎеҷЁдёәдәҶйҒҝе…Қе®•жңәзҡ„йЈҺйҷ©, йғҪдјҡжҠҠжңҚеҠЎеҷЁжӢҶзҡ„зү№еҲ«з»Ҷ, дҪҶжҳҜиҝҷе°ұдјҡеёҰжқҘдёҖдёӘй—®йўҳ, жң¬жқҘдёҖд»¶дәӢжғ…иғҪеңЁдёҖдёӘең°ж–№йӣҶдёӯеӨ„зҗҶе®Ң, зҺ°еңЁиҰҒеӨҡиө°еҮ жқЎй“ҫи·ҜжүҚиғҪеҒҡе®Ң, ж•ҲзҺҮдјҡеҸҳдҪҺ, еҰӮжһңеҶҚеҠ е…ҘдёҖдёӘ redis, й“ҫи·ҜеҸӘдјҡжӣҙй•ҝ

дёҚиҝҮ, жҲ‘д»¬е®һзҺ°зҡ„ж—¶еҖҷеҸҜд»ҘеҸӮиҖғ redis зҡ„ SkipList.

жҺ’иЎҢжҰңжңҚеҠЎе®һзҺ°

йҰ–е…Ҳ жҲ‘д»¬иҰҒжҠҪиұЎеҮә key е’Ң value иҝҷдёӨдёӘжҰӮеҝө, key дҪ еҸҜд»ҘзҗҶи§Јдёә playerID, йӮЈд№ӢжүҖд»ҘеҸ« key жҳҜеӣ дёә жңүзҡ„жҺ’иЎҢжҰңе®ғдёҚжҳҜд»Ҙи§’иүІдёәеҚ•дҪҚ, е®ғд№ҹжңүеҸҜиғҪжҳҜд»Ҙе® зү©дёәеҚ•дҪҚ, жӯӨж—¶зҡ„ key е°ұжҳҜ petID. value е°ұжҳҜеҲҶеҖј, дёҚиҝҮиҝҷйҮҢдёҚиғҪз®ҖеҚ•ең°е°ұдёҖдёӘеҲҶеҖј, иҜ•жғідёҖдёӢ, еҰӮжһңеҲҶеҖјзӣёеҗҢдәҶжҖҺд№ҲеҠһ?

д»ҘдёӢд»Јз ҒеқҮдёәдјӘд»Јз Ғ, йҡҸжүӢдёҖеҶҷ.

Data иЎЁзӨәзҡ„е°ұжҳҜ иҝҷдёӘ key з”ЁдәҺжҺ’еәҸзҡ„дҝЎжҒҜ, йӮЈд№Ҳиҝҷж—¶еҖҷжңүдёӘй—®йўҳ еҰӮжһң иҝҷдёӘ key еҗҢж—¶еҮәзҺ°еңЁ A, B дёӨдёӘжҺ’иЎҢжҰң, йӮЈд№Ҳ key е°ұеҸҜиғҪйҖ жҲҗж•°жҚ®еҶ—дҪҷ, еҚ з”ЁеҶ…еӯҳ, иҝҷдёӘж—¶еҖҷ е°ұиҰҒж”№йҖ  Data иҝҷдёҖ з»“жһ„.

иҝҷж—¶еҖҷ еҰӮжһңдёҖдёӘ key еҮәзҺ°еңЁеӨҡдёӘжҺ’иЎҢжҰң еҸӘйңҖиҰҒеңЁ иҝҷдёӘ map йҮҢ йҖҡиҝҮ жҺ’иЎҢжҰңID еҺ»жүҫеұһдәҺиҝҷдёӘжҺ’иЎҢжҰңзҡ„еҲҶеҖје°ұиЎҢдәҶ.

жӯӨж—¶еҸҲеҮәзҺ°дәҶдёҖдёӘй—®йўҳ, player key е’Ң pet key жңүеҸҜиғҪзӣёеҗҢ йҖҡиҝҮ key жқҘжҹҘжүҫж•°жҚ® жҳҫз„¶жҳҜдёҚеҗҲзҗҶзҡ„.

иҝҷж—¶еҖҷе°ұиҰҒеј•е…ҘдёҖдёӘ DataType еҸҜд»ҘзҗҶи§Јдёә key зҡ„зұ»еһӢ, е®ғдёҺ жҺ’иЎҢжҰңLeaderBoard з»‘е®ҡеңЁдёҖиө·.

иҝҷж ·е°ұиғҪйҖҡиҝҮ жҺ’иЎҢжҰң еҸҚжҹҘеҲ°е…¶ж——дёӢзҡ„жҺ’иЎҢдҝЎжҒҜ.

и®Іжё…жҘҡдәҶжҺ’иЎҢжҰңзҡ„и®ҫи®Ў, жҺҘдёӢжқҘе°ұжҳҜйҮҚеӨҙжҲҸ SkipList зҡ„е®һзҺ°, иҝҷйҮҢжҲ‘ жӢҝеҮә Redis зҡ„жәҗз ҒжқҘи®Іи§Ј.

SkipList еҺҹзҗҶ

е…¶е®һдҪ еҸҜд»Ҙиҝҷд№ҲзҗҶи§Ј, жҷ®йҖҡзҡ„й“ҫиЎЁ жҗңзҙўдёҖдёӘдёңиҘҝ иҰҒйҒҚеҺҶдёҖж¬Ў ж—¶й—ҙеӨҚжқӮеәҰдёә O(n), иҰҒжғіжҸҗй«ҳжҗңзҙўйҖҹеәҰ, е°ұиҰҒе°ҪеҸҜиғҪи·іиҝҮеӨҡзҡ„иҠӮзӮ№, йӮЈд№Ҳз»ҷе®ғе»әеӨҡжқЎзҙўеј•й“ҫе°ұиЎҢдәҶ.

д»ҺжңҖдёҠйқўзҡ„зҙўеј•й“ҫиЎЁејҖе§Ӣжҗң 30 зҡ„жҺ’еҗҚ еҸҜд»ҘеғҸдёӢйқўиҝҷж ·жҗңзҙў

min еҗҺйқў иҰҒи·Ёи¶Ҡ span = 3 жүҚиғҪеҲ° 7, 7иҰҒи·Ёи¶Ҡ span = 1 жүҚиғҪеҲ° 30, еӣ жӯӨ 30 д»Һе°ҸеҲ°еӨ§дёәз¬¬еӣӣеҗҚ.

иҮіжӯӨ, жҲ‘д»¬зҹҘйҒ“ skiplist жң¬иҙЁе°ұжҳҜеӨҡдёӘзҙўеј• еҝ«йҖҹжҹҘжүҫ, еҗҢж—¶з»ҙжҠӨдәҶдёҖдёӘ spanеҖј, жқҘеҝ«йҖҹеҫ—еҲ° жҺ’еҗҚ

Redis дёӯ SkipList зҡ„е®һзҺ°

йҰ–е…ҲжҳҜиҰҒзЎ®е®ҡ, жҲ‘д»¬жҸҗеҚҮдёҖдёӘиҠӮзӮ№еҲ°зҙўеј•й“ҫзҡ„жҰӮзҺҮ, е’Ң иҰҒе»әеӨҡе°‘жқЎзҙўеј•й“ҫ, redis дёӯ жңҖеӨ§еұӮзә§дёә32еұӮ, жҜҸж¬ЎжҸҗеҚҮиҠӮзӮ№зҡ„жҰӮзҺҮдёә 0.25.

йҖҡиҝҮиҝҷдёӘйҡҸжңәз®—жі•, еҸҜд»ҘзІ—з•Ҙзҡ„и®Ўз®—

  • еұӮж•°дёә 1, жҰӮзҺҮдёә 1 - p
  • еұӮж•°дёә 2, жҰӮзҺҮдёә (1 - p)*p
  • еұӮж•°дёә 3, жҰӮзҺҮдёә (1 - p)pp

е№іеқҮеұӮж•°дёә (1-p)+ (1-p)p + (1-p)p*p + вҖҰ = 1 / 1 - p

еҰӮжһңжҢүз…§ redis зҡ„жҸҗеҚҮжҰӮзҺҮ 0.25 жқҘз®—, йӮЈд№Ҳз©әй—ҙдёҠиҶЁиғҖ (4 / 3) еҚі 1.33еҖҚ.

дёӢйқўжқҘзңӢ SkipList зҡ„е®ҡд№ү.

Insert жҸ’е…Ҙж“ҚдҪң

дё»иҰҒжҳҜд»ҺжңҖй«ҳеұӮ, дҫқж¬Ўеҗ‘дёӢжүҫ ж–°жқҘзҡ„еҖјеә”иҜҘжҸ’е…ҘеҲ°е“Ә, з„¶еҗҺ дҝқеӯҳеҲ° update иҝҷдёӘж•°з»„дёӯ, иҝҷж ·е°ұдёҚз”ЁжҜҸжүҫдёҖеұӮе°ұж”№дёҖеұӮ.
